# AI Chatbot

## What does it do?

This project is a REST API built using **Node.js** and **Express.js**. It provides endpoints that allow clients to send HTTP requests and receive JSON responses. The API is organized using a modular structure with separate folders for controllers, routes, and data, making the code easier to maintain and extend. It can be tested using tools like Postman or Thunder Client.

## How to Run

### 1. Install dependencies

```bash
npm install
```

### 2. Start the server

```bash
node server.js
```

Or, if your project uses **nodemon**:

```bash
nodemon server.js
```

### 3. Open the API

By default, the server runs at:

```
http://localhost:3000
```

You can now test the API endpoints using **Postman**, **Thunder Client**, or your web browser (for GET requests).
